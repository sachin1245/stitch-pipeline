# Kinetic Precision — Demo Project

A crypto analytics dashboard built entirely through the stitch-pipeline, demonstrating the full design-to-code workflow.

## Tech Stack

- React 19 + TypeScript
- Vite 8
- Tailwind CSS v4
- React Router DOM

## Design System: Ether Flow

- **Mode**: Dark / Vibrant
- **Primary**: `#00FFA3` (mint green)
- **Secondary**: `#00E5FF` (cyan)
- **Tertiary**: `#FF3B69` (coral pink)
- **Headlines**: Space Grotesk
- **Body**: Manrope
